# Mito

A wireless split mechanical keyboard with rgb

## WIP

Many parts of this project are still being actively worked on

## Features

- Split
- Wireless
- Per key RGB backlight
- Low-profile khail choc switches
- hot-swap compatible
- transparent PETG case and plate
- ZMK
- NRF52833 soc
